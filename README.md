# Trabalho 1 - Configuração do Ambiente e Hello World em Dart

**Nome Completo:** Luis Gustavo Moraes Kadanus  
**Objetivo:** Configurar o ambiente de desenvolvimento Dart e executar um programa simples que imprima "Hello World" no terminal.

## Instruções para executar o programa
1. Instale o Dart SDK (ou Flutter SDK).
2. Abra o terminal na pasta do projeto.
3. Execute o comando:
   ```bash
   dart run main.dart
   ```

## Exemplo da Saída
Ao executar o comando acima, o terminal deve exibir:
```bash
Hello World
```
